# CodeAlpha Handwritten Character Recognition

## Project Overview
This project is developed as part of the CodeAlpha Machine Learning Internship.

The objective is to recognize handwritten digits using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset
- MNIST Handwritten Digits Dataset

## Model
- Convolutional Neural Network (CNN)

## Features
- Image preprocessing
- Digit classification
- Model training and evaluation
- High prediction accuracy

## Results
- Achieved approximately 99% accuracy on the test dataset.
